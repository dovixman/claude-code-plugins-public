# Daily Notes

```bash
obsidian daily [paneType=tab|split|window]
obsidian daily:path
obsidian daily:read
obsidian daily:append content=<text> [paneType=tab|split|window] [inline] [open]
obsidian daily:prepend content=<text> [paneType=tab|split|window] [inline] [open]
```

- `daily` opens today's daily note.
- `daily:path` returns expected path even if file doesn't exist yet.
- `inline` suppresses leading newline.
