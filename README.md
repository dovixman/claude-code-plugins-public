# agent-bridge

A Claude Code plugin for bidirectional messaging between Claude Code sessions via MCP channel notifications.

## Plugin

| Plugin | Description |
|--------|-------------|
| [agent-bridge](./agent-bridge) | Enables multiple Claude Code sessions to communicate in real-time through a shared filesystem mailbox |

See [agent-bridge/README.md](./agent-bridge/README.md) for full documentation, setup instructions, and usage.
